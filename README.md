# Orphanage Fund
----Basic PHP project----- 
-For windows
  * Install XAMPP or Wamp server
  * Keep the project on this server
  * Install mysql
  * Create database 'of'

-For linux
  * Install Apache
  $ sudo apt install apache2
  $ sudo systemctl status apache2
  $ cd /var/www/html
  Keep the project and run
  * Install mysql
  * Create database 'of'
  $ sudo apt install mysql-server
